# [JoinSystem](https://github.com/syoooooooooyu/JoinSystem)

プレイヤーが初inしたときに任意のpasswordを入力しないと動けないプラグインです

# 利用方法

syouyu/join/form/CustomForm.phpの23行目``if($data[0] == "ぎょうざ")``のぎょうざを任意のパスワードにしてください。(それがpasswordになります)
